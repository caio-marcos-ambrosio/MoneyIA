# MoneyIA

Este é um repositório que contém o código de um assistente financeiro desenvolvido como introdução a ChatBots nas aulas de Inteligência Artificial do professor Marcelo Grilo para o 2º ano do Instituto J&F.

## Tecnologias utilizadas

* **Python** - linguagem principal do projeto.
* **LangChain** - framework utilizado para integração com modelos de linguagem e criação das cadeias de processamento do assistente.
* **LangGraph** - biblioteca utilizada para modelar e orquestrar o fluxo de execução do chatbot por meio de grafos de estados.
* **PostgreSQL** - banco de dados relacional para armazenamento das transações financeiras e eventos.
* **Git e GitHub** - controle de versão e hospedagem do código.

## Funcionalidades

* Registro de receitas e despesas.
* Consulta do saldo atual e em um período específico.
* Classificação de gastos por categoria.
* Auxílio na educação financeira.
* Interação com o usuário por meio de linguagem natural.
* Persistência e organização dos dados financeiros.

## Objetivo

O projeto tem como objetivo apresentar os conceitos básicos de ChatBots e Inteligência Artificial por meio da construção de um assistente capaz de auxiliar os usuários no controle de suas finanças pessoais, promovendo uma melhor educação financeira e demonstrando, na prática, conceitos de agentes de IA e processamento de linguagem natural.
